# New Product Playbook

1. Write an idea brief.
2. Evaluate the idea.
3. Write product strategy.
4. Write requirements.
5. Write ADRs for exceptions.
6. Generate product starter:

```bash
npm run init:product -- products/my-product
```

7. Create an implementation plan.
8. Run the plan gate.
9. Implement.
10. Run the implementation gate.
11. Create and verify offline bundle.
